PC-ED: Testing Automated Agents to Assess the Quality of Training Resource Metadata

We developed an automated agent system using LLMs and RAG to evaluate and improve the quality of CCIHE training resource metadata, dynamically updating the database with new Q&A pairs. Integrated into a React dashboard for secure, enhanced user interaction.

RCSB Protein Data Bank: Python Software Package for Accessing RCSB PDB APIs

A Python wrapper for RCSB PDB's data API enabling streamlined access and integration of 3D macromolecular structure data. Built with GraphQL, rustworkx, networkx, and Azure. Published open-source on GitHub and PyPI.

Read Paper → GitHub → PyPI → Journal of Molecular Biology

FIA 2026 F1 Reference Car — Procedural Parametric Model

Full procedural reconstruction of the FIA 2026 Formula 1 reference car geometry generated entirely from the technical regulations using a FreeCAD Python macro. Every volume — floor, nose, chassis, sidepods, front and rear wings, halo, wheel drums, diffuser — is built from first principles via CSG booleans and plane trims derived from regulatory dimensions.
