AeroGen

AeroGen is my dissertation research project building an autonomous aerodynamic design pipeline for Formula 1 car geometries under the FIA 2026 regulations. The first stage, now completed, is a procedural reconstruction of the FIA 2026 reference box car: a full regulation-compliant geometry encoding more than 35 reference volumes from the technical regulations using CSG boolean operations and plane trims. This envelope defines the legal boundary that every generated design must satisfy.

The next stage constructs a parametric F1 car inside this envelope, controlled by a design parameter dictionary. Sampling the parameter space programmatically generates a large dataset of regulation-compliant geometry variations, which feeds into CFD simulations to produce aerodynamic performance labels. This labeled dataset will support deep learning models for fast aerodynamic design exploration, reducing the need to run CFD at every design point.

rcsb-api

A Python package for programmatic access to the RCSB Protein Data Bank — home to over 230,000 3D structures of proteins, nucleic acids, and macromolecular machines. The package wraps the RCSB Search and Data APIs in a Pythonic interface, allowing researchers to query, retrieve, and integrate structural biology data without needing to understand the underlying GraphQL schema. Published in Journal of Molecular Biology (2025) .

HPC-ED

HPC-ED is a federated catalog that aggregates training materials — tutorials, workshops, and course modules — from partner institutions across the HPC and cyberinfrastructure community. As part of the project, I built a proof-of-concept automated quality assurance system that uses a two-stage LLM pipeline to crawl catalog entries, including YouTube transcript extraction, summarize content, and score metadata accuracy against the actual resource. The system was evaluated across GPT-3.5 Turbo, GPT-4o Mini, GPT-4.1 Nano, and GPT-4.1 Mini. Published and presented at BPHTE '25, the SC Workshop on Best Practices for HPC Training and Education.

OmniPARC

OmniPARC is a physics-aware neural surrogate model for shock-compressed solid reactive materials. The project works toward foundation models for energetic materials by learning shared dynamics across different chemical species and microstructural geometries, enabling prediction and analysis of chemistry–structure–performance relationships.

LLM Dashboard for Carnegie Classification Queries

Built an LLM and retrieval-augmented generation application for answering queries related to the Carnegie Classification of Institutions of Higher Education (CCIHE) through a React dashboard. The system retrieves relevant information, generates responses, and dynamically updates a database with new question–answer pairs, improving access to higher education classification data while supporting secure and user-friendly interaction.
